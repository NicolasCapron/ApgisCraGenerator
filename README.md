# ApgisCraGenerator

## Python Script: Generate PowerPoint Presentation for Product Owners

This repository includes a Python script to generate a PowerPoint presentation (`Presentation_ApgisPdfTools_PO.pptx`) in French, intended for Product Owners.

### Prerequisites

Install the required dependency:

```bash
pip install python-pptx
```

### Usage

Run the script from the repository root:

```bash
python ApgisCraGenerator/generate_Presentation_ApgisPdfTools_PO.py
```

### Output

The script generates `Presentation_ApgisPdfTools_PO.pptx` in the current working directory. This presentation includes:
- 16 slides covering the migration to ApgisPdfTools (PdfTools SDK)
- Speaker notes for presenters
- Topics: context, business benefits, functional overview, impacts, risks, migration plan, and acceptance criteria

Do not commit sensitive information (such as license keys or credentials) to the repository.
